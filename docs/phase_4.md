# DealWise Phase 4 Plan

# Name

Deal Score, Scam Score and Buy Decision Engine

# Goal

Phase 4 turns listings into obvious buying decisions.

Instead of only showing title, price and URL, DealWise should show:

- Deal Score
- Scam Risk
- Build Fit
- Budget Fit
- Evidence Confidence
- Final Decision

# Current Foundation

Implemented in version 0.4.0:

- Live Deals rows show deal/scam/build/evidence scores.
- Listings are labelled with a decision such as Buy Now, Negotiate, Watch, Wait or Avoid.
- Live Deals prioritises items matching the PC Builder parts checklist.
- Saved / favourited deals are separated from new worth-checking deals.

# Future Work

- Real price history.
- Seller reputation checks.
- Marketplace-specific risk rules.
- Completed listing comparisons.
- Better evidence tracking.
- Better compatibility scoring.
