# DealWise Phase 5 Plan

# Name

Image Check and Reverse Image Search Foundation

# Goal

Phase 5 starts the suspicious-image workflow for marketplace listings.

# Current Foundation

Implemented in version 0.4.0:

- Vinted image URLs are surfaced in Live Deals where available.
- Listing cards can show thumbnails.
- Image Check opens the listing image for manual inspection.
- Image downloads are cached locally under the DealWise user cache directory.

# Future Work

- Drag-and-drop image upload.
- Paste image from clipboard.
- Store image-check notes.
- Browser handoff to reverse-image providers.
- Reverse-image result model.
- Scam score integration.
- Side-by-side listing image comparison.
