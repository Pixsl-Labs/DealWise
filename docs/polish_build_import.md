# DealWise Polish + Build Import Phase

## Goal

Make DealWise feel more polished and daily-usable before adding major Phase 7 marketplace connectors.

## Added

- Build Import window for copied build lists.
- Natural-language-ish component parser.
- Reused/existing part detection.
- Automatic checklist population.
- Build summary export to Markdown and JSON.
- Clipboard build summary export.
- Hidden Deals section with restore.
- Real-time/debounced Live Deals filtering.
- Deal Score details window.
- Wishlist Progress card.
- Search Builder card.
- Smoother image loading placeholder and safer image cache extensions.

## Still Future

- Proper virtualised list rendering for very large result sets.
- Real marketplace completed-sale price history.
- Multi-marketplace search builder.
- Full reverse image search automation.
- Unit tests for parsers and scoring.
